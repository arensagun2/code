1. Provide a high-level description (ie, using minimal amount of technical
   jargon) of the project in no more than 200 words.

   This project aims to enhance the functionality of the j-- language by modifying its lexical
   and syntatic grammars using JavaCC. The changes will be done in two parts, one for scanning and one for parsing.
   Scanning part involves adding features like multiline comments, operators, reserved words, and literals.
   The parsing part will focus on adding support for the data types (long and double), operators, conditional expressions,
   loop and control statements (do-while, for, break, switch and continue statements), exception handling, 
   and interface type decleration.
